# Flashcard App - 瞬間作文練習

このアプリは、CSV形式の例文データを元にフラッシュカード形式で瞬間作文を練習するためのツールです。

## セットアップと起動

1. **依存関係のインストール**:
   ```bash
   npm install
   ```

2. **アプリとログサーバーの起動**:
   ```bash
   npm run dev:all
   ```
   - フロントエンド: `http://localhost:5174` (ポートは環境により変わる場合があります)
   - ログサーバー: `http://localhost:3001` (学習履歴の保存用)

## 教材の追加・更新手順

アプリはルートディレクトリにある `.csv` ファイルを教材として読み込みます。

### 1. CSVファイルを配置
プロジェクトのルートディレクトリ（`flashcard-app` の一つ上の階層）に、教材となるCSVファイルを配置してください。

#### 対応カラム（ヘッダー名）:
- `日本語` (必須): 表面に表示される問題
- `英語` (必須): 英語モードの正解
- `中国語` (必須): 中国語モードの正解
- `ピンイン` (任意): 中国語モードで正解の下に表示
- `英語別解` (任意): 英語モードで「別の表現」として表示
- `中国語別解` (任意): 中国語モードで「別の表現」として表示
- `補足` または `ニュアンス` (任意): 正解の下に補足情報として表示

### 2. JSONへの変換スクリプトを実行
Reactアプリで高速に読み込むため、CSVをJSONに変換する必要があります。
```bash
python3 scripts/convert_csv.py
```
- このスクリプトを実行すると、`src/data/` 内に各教材のJSONファイルと、リストを管理する `materials.json` が自動生成されます。
- アプリを起動中の場合は、変換後にブラウザをリロードすると新しい教材が反映されます。

## 学習履歴の管理
学習を完了（最後のカードで Check/Finish をクリック）すると、以下のファイルに履歴が追記されます。
- `../logs/study_history.json`

このプランには経過時間、教材名、言語が含まれており、成長の記録として活用できます。

## キーボードショートカット
- `Space`: カードをめくる（正解を表示/非表示）
- `→` (右矢印): 次のカードへ
- `←` (左矢印): 前のカードへ
